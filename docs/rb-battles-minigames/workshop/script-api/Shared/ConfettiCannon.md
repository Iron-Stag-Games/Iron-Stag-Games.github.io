# ConfettiCannon

A single-method module for spawning confetti particles.

```lua
local ConfettiCannon = require(ReplicatedStorage.SharedModules.ConfettiCannon)
```

## Methods

### Create

Spawns a confetti particle.

```lua
ConfettiCannon.Create(position: Vector3, direction: Vector3, totalRibbons: number): ()
```
